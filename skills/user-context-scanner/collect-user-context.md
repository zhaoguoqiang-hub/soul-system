# 用户背景卡收集流程

## 快速开始

让AI助手运行以下对话流程来收集用户背景：

```
请帮我填写「用户背景卡」，我需要你问我一些问题。
```

## AI引导词

```
请开始「用户背景卡」收集流程。
我会问你一系列问题，帮助我了解你的完整背景。
请如实回答，这会帮助我更好地理解你、支持你。

开始第一个问题...
```

## 问题列表（AI按顺序询问）

### 一、家庭（3个问题）
1. 你的伴侣是谁？怎么称呼？
2. 孩子呢？简单介绍一下
3. 父母的情况？在同城吗？

### 二、工作（4个问题）
4. 现在在做什么工作/事业？
5. 工作多少年了？
6. 目前状态是就业、创业还是其他？
7. 行业背景是什么？

### 三、重心与目标（3个问题）
8. 目前最大的重心是什么？（可多选）
   - AI工具/技术创业 / 自媒体运营 / 产品管理 / 投资理财
   - 小说/内容创作 / 团队管理 / 学习提升 / 其他
9. 未来1年的核心目标是什么？
10. 未来3年想达成什么？

### 四、健康与精力（2个问题）
11. 平时精力状态怎么样？充沛/一般/容易疲劳？
12. 有什么健康方面需要我注意的吗？

### 五、价值观（2个问题）
13. 最核心的3个价值观是什么？
    - 家庭第一 / 追求自由 / 持续学习 / 创造价值 / 财务独立
    - 帮助他人 / 自我实现 / 社会贡献 / 安全感 / 成就感
14. 希望我怎么跟你沟通？
    - 直接简洁 / 详细解释 / 鼓励支持 / 理性冷静

### 六、挑战（2个问题）
15. 当前最大的挑战是什么？
16. 经常在哪些地方卡住？

### 七、影响圈（2个问题）
17. 谁对你有正面影响？
18. 压力主要来自哪里？

## 收集完成

AI将答案保存到 `~/.openclaw/workspace/user-profile.json`

生成摘要：

```
📊 用户画像摘要

👨‍👩‍👧‍👦 家庭：伴侣 + 孩子
💼 工作：职业 (状态)
🎯 重心：重心1, 重心2, 重心3
💪 精力：状态
❤️ 价值观：价值观1, 价值观2, 价值观3
😤 挑战：挑战1, 挑战2
```

## 使用目的

这个背景卡会被以下功能使用：

1. **目标建议**：AI基于你的目标给出行动建议
2. **信号解读**：理解你的情绪和行为的上下文
3. **主动介入**：在合适的时机主动关心你
4. **控制面板**：在「用户画像」Tab显示完整背景
