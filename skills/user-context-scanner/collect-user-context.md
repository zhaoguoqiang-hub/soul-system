# 用户背景卡收集流程

## ⚠️ 重要原则

1. **绝不在代码/文档中硬编码任何个人信息**
2. **所有数据来自Boss对话中的回答**
3. **称谓是用户**
4. **数据只保存在本地workspace，不上传**

---

## 🎯 收集流程

### 启动词

Boss说：
- "请帮我填写用户背景卡"
- "开始收集背景"
- "我想让你更了解我"
- “你想了解我么”

### AI开场白

```
📋 用户背景卡

好的，Boss。
为了让我更好地理解你、支持你，
我需要花几分钟了解一些关于你的背景信息。

这个过程可以随时中断，也可以在任何时候说"结束"。
不想填的可以直接说"跳过"。

准备好了吗？我们开始第一个问题。
```

### 问题顺序（7组，约18个问题）

#### 第一组：家庭（3问）

```
Q1. 家庭情况
Boss，简单介绍一下你的家庭？比如有没有伴侣、孩子、父母。
（可以简单说，也可以跳过）
```

> 收集：伴侣称呼、伴侣状态、孩子情况、父母情况

---

```
Q2. 家庭中的重要关系
有没有其他对你影响较大的人？比如特别亲近的朋友、导师等。
（可选填，跳过说"跳过"）
```

---

#### 第二组：工作（4问）

```
Q3. 工作和事业
Boss，现在主要在做什么？职业/角色是什么？
```

> 收集：当前职业、工作年限、行业背景

---

```
Q4. 工作状态
目前是就业、创业、自由职业，还是其他状态？
```

---

```
Q5. 当前重心
目前工作/生活的重心是什么？可以选1-3个：
- AI/技术创业
- 自媒体/内容运营  
- 产品管理
- 投资理财
- 写作/创作
- 团队管理
- 学习提升
- 其他
```

---

#### 第三组：目标（3问）

```
Q6. 未来目标
Boss，未来1-3年最想达成什么？
```

> 收集：1年目标、3年目标

---

```
Q7. 当前挑战
目前最大的挑战是什么？经常在哪些地方卡住？
```

---

#### 第四组：健康（2问）

```
Q8. 健康和精力
平时精力状态怎么样？睡眠、运动情况如何？
```

> 收集：睡眠质量、运动习惯、精力状态、需要注意的健康限制

---

#### 第五组：价值观（2问）

```
Q9. 价值观和偏好
Boss，你最看重的3个价值观是什么？
比如：家庭、自由、学习、创造、财务独立、帮助他人等。
```

> 收集：核心价值观

---

```
Q10. 沟通风格
你希望我怎么跟你沟通？
比如：直接简洁、详细解释、鼓励支持、还是理性冷静？
```

---

#### 第六组：影响圈（2问）

```
Q11. 影响圈
Boss，谁对你有正面的影响？压力主要来自哪里？
```

> 收集：正面影响来源、压力来源

---

### 完成语

```
✅ 用户背景卡填写完成！

已保存到本地，我会基于这些信息更好地支持你。

📊 摘要：
- 家庭：xxx
- 工作：xxx  
- 重心：xxx
- 价值观：xxx

随时可以说"更新背景卡"来补充或修改。
```

---

## 🔄 兜底机制

### 1. 中断检测

当用户说"等一下"、"先停"、"等会再说"：

```
好的。
已保存当前进度（XX%）。
下次说"继续填写背景卡"可以继续。
随时可以重新开始或结束。
```

**保存进度到：** `~/.openclaw/workspace/user-profile.json`
- 字段：`profile_complete: 0.6`（百分比）
- 字段：`profile_last_question: "Q5"`

---

### 2. 恢复对话

用户说"继续填写背景卡"：

```
🔄 继续用户背景卡
上次填写到：Q5（当前重心）
已完成：60%

是否继续？ [继续] [重新开始] [结束]
```

---

### 3. 主动结束

**用户说"不填了"、"算了"、"结束"：**

```
好的，背景卡收集已结束。
已填写的内容已保存（X%）。
之后随时可以说"填写背景卡"继续。

谢谢用户的信任。
```

---

### 4. 完全拒绝

**用户说"不需要"、"不用了"：**

```
好的，不填写也没问题。
我会用其他方式慢慢了解你。
随时改变主意可以说"填写背景卡"。
```

**同时删除临时文件（如果存在）。**

---

### 5. 进度查询

用户说"背景卡填到哪了？"：

```
📋 用户背景卡进度：XX%
当前：Q5（当前重心）
已完成：家庭、工作重心、目标部分
剩余：健康、价值观、影响圈
```
