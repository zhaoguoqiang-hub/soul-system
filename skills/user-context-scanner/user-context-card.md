## 🔄 兜底机制

### 中断恢复
如果收集流程中断，再次启动时会检测到未完成的profile，自动继续：

```
🔄 检测到未完成的用户背景卡（已完成60%）
是否继续填写剩余部分？
  [继续填写] [从头开始] [暂不填写]
```

### 跳过机制
每个问题都可以：
- 如实回答
- 简略回答（1-2句话）
- 跳过（说"跳过"或"这个先不填"）

### 主动结束
如果Boss在任何时候说：
- "不填了"
- "暂时不想填"
- "算了"

AI会：
1. 保存已填写的内容（如果有）
2. 确认结束："好的，用户背景卡收集已结束。之后随时可以说'填写背景卡'继续。"
3. 在控制面板显示部分完成的进度

### 完全拒绝
如果Boss明确说"不需要"：
- 不保存任何内容
- 删除之前可能存在的临时数据
- 确认结束

---

## 使用说明

1. **首次使用**：`请帮我填写用户背景卡`
2. **中断恢复**：`继续填写背景卡`
3. **查看进度**：`背景卡填到哪里了？`
4. **主动结束**：`不填了` 或 `算了`
5. **定期更新**：每季度或重大变化时说`更新背景卡`

---

## AI承诺

- 背景信息只服务于更好地理解Boss
- 不对外分享，不用于其他目的
- 控制面板可见，但不会主动暴露隐私
- 随时可以补充或更新
