## 使用说明

1. **首次使用**：请完整填写本模板，帮助AI建立对你的基础认知
2. **定期更新**：每季度或重大变化时更新一次
3. **补充细节**：可以在填写时加入更多个人化的细节描述

## AI承诺

- 你的背景信息只服务于更好地理解你
- 不对外分享，不用于其他目的
- 控制面板可以看到，但不会主动暴露你的隐私
