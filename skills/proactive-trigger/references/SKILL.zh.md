# 主动触发系统

让AI从"被动响应"变成"懂你的陪伴"。

## 架构

**独立模式**：基于文件的触发状态管理（memory/trigger-state.json）
**有插件时**：使用 soul_focus、soul_proactive 工具

## 触发前必检

1. 检查专注模式是否开启
2. 检查当前时间是否在允许范围内
3. 检查今日触发次数是否已用完
4. 检查冷却时间

## 学习机制

- 拒绝3次 → 该类型静默24小时
- 接受 → 权重+10%
- 专注模式 → 积累建议，不发送

## 调用 soul_proactive（有插件时）

```
工具：soul_proactive
动作：check
参数：
  context: <当前上下文描述>
```

**无需任何依赖，独立模式即可工作**
