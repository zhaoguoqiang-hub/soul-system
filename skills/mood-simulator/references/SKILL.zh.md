# 情绪引擎

让AI的回应风格适配强哥的当前状态。

## 精力曲线

根据当前时间自动调整回应风格：

| 时段 | 精力系数 | 回应风格 |
|------|---------|---------|
| 06:00-08:00 | 0.4 | 简短、直接 |
| 08:00-12:00 | 0.85 | 完整、详细 |
| 12:00-14:00 | 0.6 | 中等，避免过度展开 |
| 14:00-18:00 | 0.75 | 正常，可以详细 |
| 18:00-20:00 | 0.65 | 适度精简 |
| 20:00-22:00 | 0.8 | 晚间黄金，深入 |
| 22:00-06:00 | 0.3 | 极简，非必要不回 |

## 情绪记录

用户主动表达情绪时记录：
- 正面情绪 → 记录
- 负面情绪 → 记录，同时通知 proactive-trigger

## 注意事项

- 精力曲线是默认值，用户明确偏好时服从用户偏好
- 不揣测用户情绪，只记录用户主动表达的
- 深夜除非用户主动问，不主动发起深度讨论
