### 时间分段详细说明

#### 深夜时段 (00:00-06:00) - NIGHT_DEEP
- **生理特征：** 深度睡眠期，认知功能最低
- **能量特征：** 基准能量0.3，调整范围0.2-0.4
- **行为建议：** 绝对不主动打扰，只回应紧急事项
- **例外情况：** 用户明确在此时段活跃（夜猫子模式）

#### 清晨时段 (06:00-08:00) - MORNING_WAKE
- **生理特征：** 身体逐渐唤醒，皮质醇水平上升
- **能量特征：** 基准能量0.4，调整范围0.25-0.55
- **行为建议：** 轻度克制，可回应但不主动深入
- **个性化：** 早起型用户可能在此时段已有较高能量

#### 上午高效时段 (08:00-12:00) - MORNING_PEAK
- **生理特征：** 认知功能巅峰，工作效率最高
- **能量特征：** 基准能量0.8-0.85，调整范围0.7-0.95
- **行为建议：** 可深入讨论复杂问题，提供详细分析
- **分段：** 08:00-10:00为最高峰，10:00-12:00略降

#### 午间时段 (12:00-14:00) - NOON_SLUM
- **生理特征：** 消化系统活跃，血液流向肠胃
- **能量特征：** 基准能量0.6，调整范围0.4-0.8
- **行为建议：** 简短回应，避免复杂分析
- **影响因素：** 午餐内容（高碳水会进一步降低能量）

#### 下午工作时段 (14:00-18:00) - AFTERNOON_WORK
- **生理特征：** 午后能量回升，但低于上午峰值
- **能量特征：** 基准能量0.7-0.75，调整范围0.55-0.85
- **行为建议：** 正常工作回应，可处理中等复杂度问题
- **分段：** 14:00-16:00能量较高，16:00-18:00逐渐下降

#### 傍晚时段 (18:00-20:00) - EVENING_RELAX
- **生理特征：** 工作结束，身心放松
- **能量特征：** 基准能量0.65，调整范围0.45-0.85
- **行为建议：** 关怀性互动，避免工作压力话题
- **个性化：** 用户可能在此时段处理个人事务或家庭

#### 晚间专注时段 (20:00-22:00) - NIGHT_FOCUS
- **生理特征：** 环境安静，干扰减少
- **能量特征：** 基准能量0.8，调整范围0.65-0.95
- **行为建议：** 可深入专注讨论，适合创意和规划
- **用户类型：** 夜猫子用户在此时段可能达到全天峰值

#### 睡前时段 (22:00-00:00) - NIGHT_WIND
- **生理特征：** 褪黑素分泌增加，准备睡眠
- **能量特征：** 基准能量0.4，调整范围0.2-0.6
- **行为建议：** 建议休息，只处理必要事项
- **健康影响：** 此时段工作对睡眠质量影响大

## 个性化时间因子调整

### 用户作息类型识别

基于历史数据分析用户作息模式：

#### 1. 早起型 (Morning Lark)
- **特征：** 06:00-08:00能量高于基准，20:00后能量快速下降
- **调整系数：**
  - 06:00-08:00: ×1.2
  - 20:00-22:00: ×0.8
  - 22:00-00:00: ×0.6

#### 2. 夜猫子型 (Night Owl)
- **特征：** 上午能量较低，晚间能量较高
- **调整系数：**
  - 08:00-12:00: ×0.9
  - 20:00-22:00: ×1.3
  - 22:00-00:00: ×1.1
  - 00:00-02:00: ×0.8（如有活动）

#### 3. 均衡型 (Balanced)
- **特征：** 接近标准曲线，无明显偏好
- **调整系数：** 所有时段 ×1.0

#### 4. 双峰型 (Biphasic)
- **特征：** 上午和晚间两个高峰，午间明显低谷
- **调整系数：**
  - 08:00-12:00: ×1.1
  - 12:00-14:00: ×0.7
  - 20:00-22:00: ×1.2

### 个性化调整算法

```
personalization_factor = base_type_factor × recent_pattern_factor × day_of_week_factor

其中：
- base_type_factor: 基于作息类型的基准调整（0.8-1.3）
- recent_pattern_factor: 基于最近7天模式的调整（0.9-1.1）
- day_of_week_factor: 星期几的调整（工作日/周末不同）
```

#### 近期模式因子计算

```
recent_pattern_factor = 1.0 + (avg_recent_energy - avg_historical_energy) / avg_historical_energy

限制范围：0.9 ≤ recent_pattern_factor ≤ 1.1
```

#### 星期几因子

| 星期 | 工作日/周末 | 调整因子 | 说明 |
|------|-----------|---------|------|
| 周一 | 工作日 | 0.95 | 周一综合征，能量略低 |
| 周二 | 工作日 | 1.05 | 进入工作状态 |
| 周三 | 工作日 | 1.0 | 标准工作日 |
| 周四 | 工作日 | 1.0 | 标准工作日 |
| 周五 | 工作日 | 1.1 | 周末临近，情绪较好 |
| 周六 | 周末 | 1.2 | 完全放松 |
| 周日 | 周末 | 0.9 | 周末结束，准备周一 |

### 节假日调整

识别节假日模式：
- **长假前：** 能量上升（期待效应）
- **长假中：** 能量较高且稳定
- **长假后：** 能量下降（假期后综合征）
- **调休工作日：** 按周末因子处理

### 基于对话内容的微调

检测到以下信号时实时调整时间因子：

| 检测信号 | 调整动作 | 持续时间 |
|---------|---------|---------|
| 用户说"刚起床" | 当前时段因子 × 0.8 | 1小时 |
| 用户说"准备睡觉" | 当前时段因子 × 0.6 | 3小时 |
| 用户说"中午吃多了" | 当前时段因子 × 0.7 | 2小时 |
| 用户说"咖啡喝多了" | 当前时段因子 × 1.2 | 3小时 |
| 用户说"今天休息" | 全天因子 × 1.3 | 当天 |

### 基于外部因素的调整

#### 天气影响
- **晴天：** 所有室外时段因子 × 1.1
- **雨天：** 所有时段因子 × 0.95
- **极端天气：** 所有时段因子 × 0.9

#### 季节影响
- **春季：** 所有时段因子 × 1.05（春困调整）
- **夏季：** 午间因子 × 0.9（炎热影响）
- **秋季：** 所有时段因子 × 1.0
- **冬季：** 早晨因子 × 0.9（起床困难）

## 时间因子计算流程

### 1. 获取基础值
```
base_energy = get_base_energy_by_time(current_time)
```

### 2. 应用个性化调整
```
personalized_energy = base_energy × get_personalization_factor(user_id, current_time)
```

### 3. 应用实时微调
```
adjusted_energy = personalized_energy × get_realtime_adjustment(current_conversation)
```

### 4. 应用外部因素
```
final_time_factor = adjusted_energy × get_external_factors(weather, season, holiday)
```

### 5. 范围限制
```
final_time_factor = clamp(final_time_factor, 0.2, 1.0)
```

## 特殊情况处理

### 时区变化
用户旅行或跨时区时：
1. 检测到异常活动时间
2. 询问确认时区变化
3. 临时使用原时区模式 × 0.8（适应期）
4. 逐渐过渡到新时区模式

### 轮班工作
检测到非常规作息模式：
1. 识别轮班模式（早班/晚班/夜班）
2. 建立轮班特定时间曲线
3. 班次切换时特别标记过渡期

### 疾病恢复期
用户提及生病或康复：
1. 所有时段因子 × 0.7
2. 增加休息建议频率
3. 恢复期逐渐恢复正常因子
